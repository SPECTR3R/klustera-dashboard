# Technical Test App for Klustera

This project is a technical test to apply for a developer position at [Klustera.com](https://www.Klustera.com)

## Requirements

To run this project, please:

- Fork this repo
- Clone this repo
- In the terminal, run:  

  ```
  yarn install
  yarn start
  ```
